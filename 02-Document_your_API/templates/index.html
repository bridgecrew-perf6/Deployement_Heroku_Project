API Documentation

Welcome to the API documentation for our salary prediction AP-AI.

What is AP-AI? It's a new beautiful AI model wrap into an API to help you to build the best products.

Endpoints

POST /prediction

This endpoint allows you to make a prediction of the salary amount (in dollars) according to the year.

How to request this endpoint?

You should request this endpoint using a POST method with a JSON content. Your JSON should contain at least "year" key with associated value (float).

Example: curl -i -H "Content-Type: application/json" -X POST -d '{"year": 4}' http://localhost:5000/predict

It returns a JSON like so: 

          
  {
    "prediction": "62962.191754786785"
  }

You can add rounded key with string value True in order to round the result: 

        
  $ curl -i -H "Content-Type: application/json" -X POST -d '{"year": 4, "rounded": "True"}' http://localhost:5000/predict
  {
    "prediction": "62962.0"
  }

Errors

Our API will return an error with format: 

          
  {
    "code": HTTP status code,
    "name": name of this error,
    "description": a short description
  }

Identified errors:

  • Missing JSON
  • Missing key "year"
